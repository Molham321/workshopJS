Aufgabe: Ermögliche die Eingabe von 2 Zahlen welche, einen vom Nutzer vorher definierten, Abstand haben

  1. Erstelle ein Hinweisbox
    • Sie ist am Anfang nicht sichtbar
    • Wird später eine falsche Eingabe getätigt, kann sie mit einer individuellen Warnung gefüllt werden
    • Die Hinweisbox kann geleert und ausgeblendet werden
    • Mehrere Warnungen können gleichzeitig angezeigt werden
  2. Erstelle 2 Inputfelder, die nebeneinander liegen
  3. Wird in eine der beiden Felder ein Buchstabe oder Sonderzeichen eingetragen, erscheint eine Warnung in der Hinweisbox
  4. Erstelle ein div-Element für Optionen mit folgenden Auswahlmöglichkeiten
    • Minimale Eingabe für Input 1
    • Maximale Eingabe für Input 1
    • Minimale Eingabe für Input 2
    • Maximale Eingabe für Input 2
    • Die Eingabe für Input 1 muss kleiner sein als die Eingabe für Input 2
    • Die Felder können auch leer gelassen werden und werden bei der späteren Berechnung nicht berücksichtigt
    • Ist die Eingabe zweier Felder nicht logisch (z.B. Minimale Eingabe > Maximale Eingabe) wird der Nutzer über die Hinweisbox informiert
  5. Überlege dir selbst welche Fehler auftreten könnten und informiere den Nutzer über eine getätigte Falscheingaben.
