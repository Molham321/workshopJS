Aufgabe: Schreibe verschiedene kleine Funktionen

  1. Begrüße den Benutzer
    1. Erstelle eine Inputfeld vom Typ Text mit der Id TextInput in dem man seinen Namen eintragen kann
    2. Erstelle einen Button
    3. Wird der Button geklickt geschieht folgendes:
      1. Auslesen der TextInput
      2. Eine Alert-Nachricht mit folgendem Inhalt wird eingeblendet: "Willkommen auf der Seite, [Name]"
  2. Erstelle einen Button, der bei einem Klick folgendes tut
    1. Den Wert aus dem Feld TextInput auslesen
    2. Überprüfen, ob der Wert eine Zahl ist
    3. Wenn der Wert eine Zahl ist, wird in das Feld TextInput das Produkt der Zahl mit 2 geschrieben
    4. Wenn es keine Zahl ist, soll die Funktion "showError" mit einer passenden Fehlermeldung aufgerufen werden
  3. Schreibe die Funktion "clearError", die die Hinweisbox ErrorWarnings wieder leert
  4. Addiere zwei Zahlen
    1. Erstelle 3 Inputfelder vom Typ Number mit den Ids NumberInput1, NumberInput2 und NumberOutput
    2. Erstelle ein DropDown-Menü (select) mit den vier Grundrechenarten (+/-/×/÷)
    3. Erstelle einen Button, der bei einem Klick die Berechnung von NumberInput1 und NumberInput2 mit der vorher ausgewählten Rechenoperation in NumberOuput schreibt
    4. Arbeite auch hier mit der Hinweisbox
