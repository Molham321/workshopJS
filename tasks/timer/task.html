Aufgabe: Erstelle ein Spiel, bei dem der Nutzer begrenzte Zeit hat, um Rechenaufgaben zu lösen

  1. Erstelle das Spielfeld
    • In der oberen Hälfte des Spielfelds befindet sich ein Div, in dem die Aufgabe steht
    • Unterhalb des Divs befinden sich die verschiedenen Antwortmöglichkeiten
    • Oben rechts in der Ecke befindet sich ein Timer
    • Oben links im Feld befindet sich ein Feld für den Punktestand
  2. Über dem Spielfeld befindet sich ein Button um ein neues Spiel zu starten
  3. Startet der Spieler ein neues Spiel, passiert folgendes:
    1. Der Timer wird auf 30 Sekunden gesetzt
    2. Der Punktestand wird auf 0 gesetzt
    3. Die erste Runde wird gestartet
  4. In einer Runde kann passiert folgendes:
    • Es wird eine zufällige Rechenaufgabe gestellt:
      • Hierzu werden 2 bis 3 Zahlen mit zufälligen Rechenoperatoren (+ - / x) verknüpft
      • Bei den Antwortmöglichkeiten ist die richtige und drei Falsche zufällige Antworten dabei
    • Der Nutzer wählt mit einem Klick oder per Pfeiltasten eine Antwort aus (die erste Antwort ist die linke Pfeiltaste, die zweite die obere, usw.)
    • Ist die Antwort richtig, werden den Timer 5 Sekunden gut geschrieben
    • Ist die Antwort falsch, werden 5 Sekunden abgezogen
    • Ist der Timer abgelaufen, wird das Spiel sofort beendet
  5. Gestalte das Spiel so, dass es gut aussieht
  • score: 0
  • timer: 0
